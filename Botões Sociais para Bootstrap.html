Como usar

  1. Incluir Bootstrap e Font Awesome

    Se você ainda não fez isso, inclua o mais recente Bootstrap e Font Awesome em seu projeto.

  2. Inclua o CSS ou Menos

    Você tem duas opções para ativar os botões sociais em seu projeto: CSS baunilha ou fonte Menos. Para CSS vanilla, basta incluir o bootstrap-social.cssarquivo no seu projeto.

    Por Menos, copie-o bootstrap-social.lesspara o diretório existente do Bootstrap e importe-o para bootstrap.lessvia @import "bootstrap-social.less";. Recompile quando estiver pronto.

  3. Adicione alguns botões!

    Comece a usar os botões como faria normalmente com os botões do Bootstrap que possuem um ícone adicionando a classe relevante. Por exemplo: 

      <a class="btn btn-block btn-social  btn-twitter ">
    <span class = "fa fa-twitter"> </span> Entre no Twitter
  </a>

    Ou, se você quiser apenas o botão de ícone, use-o assim: 

      <a class="btn btn-social-icon  btn-twitter ">
    <span class = "fa fa-twitter"> </span>
  </a>
